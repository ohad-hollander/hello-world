# Plan 04-01: UFO — Spawn, Movement, Collision, and Bonus Score

## Objective

Implement the mystery UFO that periodically flies across the top of the screen (UFO-01) and awards bonus points when shot by the player (UFO-02). A new `js/ufo.js` module manages all UFO state, spawning, movement, rendering, and collision. The UFO integrates into the existing game loop via `initGame()`, `updatePlaying()`, `renderPlaying()`, and `checkCollisions()`.

## Must-Haves

- A red UFO flies across the very top of the screen (y=30) at 90 px/sec, every 15-25 seconds
- UFO spawns off-screen (from either left or right edge at random) and despawns when it exits the opposite edge
- Player bullet hitting the UFO consumes the bullet, destroys the UFO, and awards bonus points (one of: 50, 100, 150, 300)
- Score increases are reflected immediately in the HUD
- UFO timer and state reset fully on `initGame()` restart (no timer leaks)

## Context

**Canvas:** 672×768 logical pixels. DPR-scaled via `ctx.scale(dpr, dpr)` in index.html. The global `ctx` is the scaled context — draw at logical coordinates.

**Script load order in `index.html`:** `constants.js` → `input.js` → `player.js` → `aliens.js` → `bullets.js` → `shields.js` → `hud.js` → `game.js`. Add `ufo.js` after `shields.js` and before `hud.js`.

**Existing globals needed by ufo.js:**
- `playerBullet` — defined in player.js (null or {x, y, w, h, vy})
- `score` — defined in game.js
- `overlaps(a, b)` — defined in bullets.js (available globally at runtime)
- All UFO constants from constants.js

**game.js current state:**
- `gameState` values: `'playing'` | `'game_over'` | `'wave_clear'`
- `initGame()` resets all state; Phase 4 extends it with `initUFO()`
- `updatePlaying(dt)` calls `updatePlayer`, `updateAliens`, `updateBullets`, `checkCollisions`; Phase 4 adds `updateUFO(dt)`
- `renderPlaying()` renders shields, aliens, player, HUD, alien bullets, ground line; Phase 4 adds `renderUFO(ctx)`

**bullets.js `checkCollisions()` current content:**
```javascript

  checkPlayerBulletVsShields();
  checkPlayerBulletVsAliens();
  checkAlienBulletsVsShields();

```

</read_first>

<action>
Create `js/ufo.js` with the following complete content:

```javascript
// js/ufo.js
// Mystery UFO — periodic flyby at top of screen, bonus points on hit

const ufo = {
  active: false,    // is UFO currently visible/moving on screen?
  x: 0,
  y: UFO_Y,
  w: UFO_W,
  h: UFO_H,
  vx: 0,           // px/sec (positive = right, negative = left)
  spawnTimer: 0,   // seconds elapsed since last despawn
  spawnInterval: 0 // randomized seconds until next spawn (set in initUFO / despawnUFO)
};

function initUFO() {
  ufo.active = false;
  ufo.spawnTimer = 0;
  ufo.spawnInterval = randomUFOInterval();
}

function randomUFOInterval() {
  return UFO_MIN_INTERVAL + Math.random() * (UFO_MAX_INTERVAL - UFO_MIN_INTERVAL);
}

function spawnUFO() {
  // Randomly enter from left or right edge
  const goRight = Math.random() < 0.5;
  ufo.vx = goRight ? UFO_SPEED : -UFO_SPEED;
  ufo.x  = goRight ? -UFO_W : LOGICAL_WIDTH;  // start fully off-screen
  ufo.y  = UFO_Y;
  ufo.active = true;
  ufo.spawnTimer = 0;
  ufo.spawnInterval = randomUFOInterval(); // schedule next appearance
}

function despawnUFO() {
  ufo.active = false;
  ufo.spawnTimer = 0;
  // spawnInterval already set in spawnUFO — reuse for next cycle
}

function updateUFO(dt) {
  if (!ufo.active) {
    // Count down to next spawn
    ufo.spawnTimer += dt;
    if (ufo.spawnTimer >= ufo.spawnInterval) {
      spawnUFO();
    }
    return;
  }

  // Move horizontally
  ufo.x += ufo.vx * dt;

  // Despawn when fully off the opposite edge
  if ((ufo.vx > 0 && ufo.x > LOGICAL_WIDTH) ||
      (ufo.vx < 0 && ufo.x + ufo.w < 0)) {
    despawnUFO();
  }
}

function renderUFO(ctx) {
  if (!ufo.active) return;
  ctx.fillStyle = '#f00'; // classic red mystery ship
  ctx.fillRect(ufo.x, ufo.y, ufo.w, ufo.h);
}

function checkPlayerBulletVsUFO() {
  // Called from checkCollisions() in bullets.js
  if (!ufo.active || !playerBullet) return;

  // AABB collision check (overlaps() defined in bullets.js)
  if (!overlaps(playerBullet, ufo)) return;

  // Hit! Award random bonus points
  const bonus = UFO_POINTS[Math.floor(Math.random() * UFO_POINTS.length)];
  score += bonus;

  // Consume bullet and despawn UFO
  playerBullet = null;
  despawnUFO();
}
```
